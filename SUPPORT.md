# Support

Need help using or contributing to this project? Here are the best ways to get support.

## Where to ask questions
1. **GitHub Issues**
   - Use Issues for bugs, feature requests, and actionable tasks.
   - Please use the templates when available.

2. **Community support**
   - Discord: https://discord.gg/yFnxh7hur3
   - Recommended channels:
     - #geral for questions
     - #project-ideas for brainstorming

## Before you ask
To get help faster, include:
- What you expected to happen
- What actually happened
- Steps to reproduce (if relevant)
- Environment info (OS, version, logs)
- Screenshots (if UI-related)
